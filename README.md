# 网页五子棋游戏

一个基于HTML5 Canvas的网页五子棋游戏，支持双人对战。

## 游戏特色

- 🎮 经典15×15五子棋棋盘
- 🎨 精美的UI设计和动画效果
- 📱 响应式设计，支持移动设备
- ⚡ 流畅的游戏体验
- 🏆 自动判断胜负

## 游戏规则

1. 黑棋先行，玩家轮流在棋盘交叉点放置棋子
2. 先在横、竖、斜任意方向连成五子者获胜
3. 点击"重新开始"按钮可以重置游戏

## 如何运行

1. 克隆或下载此项目
2. 直接在浏览器中打开 `index.html` 文件
3. 开始游戏！

## 技术栈

- HTML5 Canvas
- CSS3 (渐变、动画、响应式设计)
- 原生JavaScript (ES6+)

## 项目结构

```
gomoku-web-game/
```
